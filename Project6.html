PROJECT 6

Data Portrait

This project is to collect a set of data for over 7 days and using these data to create an data portrait that can represent ourself. I collected my sleeping data for 10 days and used these data to create a plot graph.

REFLECTION

The data i am tracking for this assignment is my sleeping schedules for 10 days. I got this sleeping data from my phone because i have this "sleep" mode on and it has like a wind down time that i set up before the actual sleep time. And i set my sleep time to like maybe 2am and wakeup at 7:10am... So i track my sleep data and i keep it updated.

Instructions + Designs: So looking at this "plot graph" i made, the X-Axis is labled as the dates i keep tracked of the hours from 10/7 to 10/16. And if you look at the Y-Axis it is the range of hours i slept. And the blinking dots represents the sleeping hours every day base on the given dates. It's not an constant data graph because my sleep schedules are very messed up due to work load. And i just collected data off my phone [convenient and easy tracking - you can find it in your apple health app]. Then i want to design it like i am dreaming so i made these little dots like "stars" that are beaming. Also i imported a image on the back to make it more immersive.

Overall: From this assignment of collecting data, i realize i am not very healthy base on how big of a difference every night. And i am pretty sure this is not a "exact" data because sometimes i don't get up in time since after i close my alarm i just fainted back to sleep and i have no idea about that. But i really need a good rest... Also in this assignment, I gain knowledge about how to use arrays and iterations creating infinity data which really helps save a lot of time.
