Personal Background

The first "computing class" I took was AP Computer Science Principle. It wasn't really a "coding" class to me because we didn't learn any professional coding languages. But it was a very useful start for me because I did not expect to study Design and Technology in College and continue to code. When I made my choice to major in this field, I took my first official coding class "Code Craft" in my freshman year, starting with P5js and some Arduino. Even though I am still not using exactly JavaScript and Python, this class really helped me with my second-year courses. With this little start, I did not struggle as much as I thought I would, and somehow coding became something really interesting yet frustrating. The point I feel success in coding is when I finally debug something, and there is no error or bug. Then I realize later on, once you understand a little bit of coding, it becomes easier to understand other programming languages. Now that I think back, it's surprising that I somehow did it, and now I'm learning more, which is very useful because once you understand how one language functions, it's easier to use others.
